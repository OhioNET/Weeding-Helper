SQL Query Builder - PHP Class
Autor: David Regla Demaree <dreglad@gmail.com>

Documentación - Español

Esta clase genera cadenas de consulta SQL que pueden ser enviadas a un servidor de base de datos (tal como MySQL)
De esta forma el código es más limpio, sin largas cadenas difíciles de interpretar ni cadenas concatenadas.
Sólo especificas el tipo de consulta, la tabla sobre la cuál trabajar y los datos a enviar/recibir

Funciones

void __construct ( string queryType )

Esta función es el constructor, establece el tipo de consulta, puede ser "select", "insert", "update", "delete", o "query". Se pueden mezclar sin problemas mayúsculas y minúsculas.
Con el tipo "query", podemos enviar una cadena de consulta arbitraria, tal como "CREATE ...","DROP TABLE...", etc..
Ver setQuery()

void setTable ( string tableName )

Esta función establece la tabla sobre la cuál vamos a trabajar

void addColumn ( string colName )

Esta función agrega una columna a la lista de columnas que serán afectadas por la consulta

void SetValue ( string value )

Est función agrega un valor correspondiente a su columa, asegúrate de que agregues las columnas en la misma posición que los valores. La primera columna que agregues, debe de corresponder con el primer valor que agregues, y así sucesivamente, sin importar dónde y cómo los agregues. Esta es la única condición!
Por cuestiones de flexibilidad, esta función no agrega las diagonales ni ninguna secuencia de escape, ése es tu trabajo!

void setWhere ( string where )

Esta función establece la cláusula WHERE (ej. "ventas < 1000")

void setGroupBy ( string groupBy )

Esta función establece la cláusula GROUP BY

void setHaving ( string havong )

Esta función establece la cláusula HAVING

void setOrderBy ( string orderBy )

Esta función establece la cláusula ORDER BY

void setLimit ( string limit )

Esta función establece la cláusula LIMIT

void setQuery ( string query )

Esta función establece la consulta que será devuelta si se selecciona el tipo "query" al crear el objeto.

void showErrors ( bool $showErrors )

Con esta función, puedes encender y apagar el reporte de errores. Muy útil al depurar.

void buildQuery ( void )

Si no falta ningún dato para el tipo de consulta seleccionado, esta función regresa la consulta final (cadena), de otro modo, y si showErrors() está en true (por defecto), imprimirá el mensaje de error.
